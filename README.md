## Installation
Fork and clone repo:
` https://github.com/enciie/abacus-rails-project `

Install all gem dependencies:

` bundle install `

Create database scheme with migrations:

` rake db:migrate `

Start local server:

` rails s `

Access site at http://localhost:3000
